# 🌐 Projeto Git HTML CSS

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar os conceitos fundamentais de desenvolvimento web utilizando HTML e CSS, além de aplicar o versionamento de código com Git e GitHub.

O projeto simula uma página web estática, focando na estruturação correta do conteúdo e estilização visual.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Git
* GitHub

## ⚙️ Funcionalidades

* Estruturação semântica de páginas
* Estilização com CSS
* Organização de arquivos
* Versionamento com Git

## 🎯 Objetivo

Consolidar conhecimentos iniciais de:

* Desenvolvimento front-end
* Versionamento de código
* Organização de projetos
